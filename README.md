# 看见傅里叶 · See Fourier

> 从旋转的圆到思维的舞蹈 —— 一部面向**零高等数学基础**用户的交互式科普网站。

通过纯粹的几何直觉与主动探索,透彻理解傅里叶变换的本质:**看见波形 → 触碰频率 → 在频域中思考**。每一个滑块、每一次拖拽,都是数学推导的一步。

---

## ✨ 特性

- **几何直觉优先**:所有公式都是对可视化现象的精确描述,而非冷冰冰的出发点。
- **交互即推导**:拖动、点击即可亲手"发现"真理,而非被动接受。
- **视听同步**:基于 Web Audio,频率变化实时转化为声音,强化直觉。
- **分屏对比实验室**:左右独立设置波形与谐波数,实时并排对比。
- **成就系统**:探索关键交互解锁趣味徽章,进度本地持久化。
- **移动优先**:触屏左右滑动切换章节,大号交互控件。
- **可访问性**:全键盘操作、色盲友好的蓝-橙配色(不依赖红绿)、支持"减少动态效果"系统偏好。

---

## 📖 章节一览

| 章节 | 主题 | 核心交互 |
|------|------|----------|
| **序幕** | 声音的配方 | 三种乐器同奏 A4(440Hz):音高相同、波形迥异,抛出"配方"之问 |
| **第一章** | 旋转的种子 | 单圆三参数(振幅/频率/相位),垂直/水平投影成波,「虚实分离」三段开关消除虚部恐惧 |
| **第二章** | 圆舞曲 | 谐波雕塑家:1–50 个圆首尾相连雕刻波形,聚焦模式、慢动作拨盘、正交性分步推导向导 |
| **第三章** | 从「指纹」到「谱」 | 时域↔频域双向拖拽联动,「随机化相位」破坏性实验,视听同步 |
| **第四章** | 通往连续的桥梁 | 周期 T 拉伸,显微镜观察离散谱线,T→∞ 触发 ∑→∫ 飞跃 |
| **终章** | 神笔傅里叶 | 手绘任意封闭曲线 → 2D DFT → 一串旋转圆分步重绘,二维频谱图 + 一维↔二维知识桥接 |

## 🚀 快速开始

环境要求:Node.js ≥ 18(开发使用 v22)。

```bash
npm install      # 安装依赖
npm run dev      # 本地开发,默认 http://localhost:5173
npm run build    # 生产构建,产物输出到 dist/
npm run preview  # 本地预览构建产物
```

---

## 🛠 技术栈

| 领域 | 选型 | 说明 |
|------|------|------|
| 框架 | React 19 | 仅管理 UI 状态与组件,不参与逐帧动画 |
| 渲染 | Canvas 2D | 承载所有高性能动画 |
| 构建 | Vite 8 | 纯静态产物,无后端 |
| 数学 | 自研轻量 DFT | 终章曲线分解(`src/engine/fft.js`) |
| 音频 | Web Audio API | 振荡器合成 → 分析节点 → 输出 |
| 公式 | KaTeX | 公式渲染与分项高亮 |

设计要点:React 状态层与 Canvas 渲染层严格分离,动画走 `requestAnimationFrame`(经 `src/hooks/useCanvasAnimation.js` 桥接),engine 为纯命令式模块、零 React 依赖。

## 📂 项目结构

```
src/
├── main.jsx              # React 挂载入口
├── App.jsx              # 分页式叙事容器(按钮/键盘/滑动三通道导航)
├── index.css           # 全局样式 + 蓝橙配色变量
├── constants/          # 全局常量(配色、参数范围、A4=440)
├── engine/             # 命令式底层引擎(与 React 解耦)
│   ├── animationLoop.js  # rAF 循环 + 慢动作倍率
│   ├── canvas.js         # Canvas2D 初始化(DPR/resize)
│   ├── fft.js            # 复数 DFT/IDFT + 二维曲线分解
│   └── audio.js          # Web Audio 引擎
├── hooks/              # engine 与 React 的适配层
│   ├── useCanvasAnimation.js
│   └── useAudioEngine.js
├── components/         # 可复用 UI 组件(Slider / ToggleSwitch / FormulaCard)
└── chapters/           # 六个章节模块,互相解耦
    ├── index.js          # 有序 chapters 数组
    ├── ch00-prelude/ … ch05-finale-drawing/
```

> 开发约定与设计铁律见 [`CLAUDE.md`](./CLAUDE.md),完整设计方案见 [`design.md`](./design.md)。
